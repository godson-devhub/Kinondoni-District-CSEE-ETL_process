#Kinondoni Distrct CSEE results ETL
Hii ni process ya ETL ambayo inahusu;

1.Extraction
Matokeo ya wanafunzi wa kidato cha nne yamechukuliwa kutoka website ya necta kwa shule 10 
ambazo zipo wilaya ya kinondoni Dar es Salaam.
2.Transformation
Matokeo hayo yameandaliwa ili kulenga somo moja la Hesabu ili kuweza kujua ufaulu kwa hizi shule kumi.
3.Loading
Matokeo yalioandaliwa vzur yanatunzwa kweny file la csv kwaajili ya maamuzi mazur ya serikali.
